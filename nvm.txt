.nvmrc
file containing version in the project root directory or any parent directory
nvm use, nvm install, nvm exec, nvm run and nvm which will respect .nvmrc file when a version is not supplied

.nvm ls-remote
list node versions disponible

nvm install 4.2.2
install node 4.2.2

nvm use 4.2.2
use the node version 4.2.2
