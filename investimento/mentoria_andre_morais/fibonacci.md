# Fibonacci
0 + 1 = 1
1 + 1 = 2
1 + 2 = 3
2 + 3 = 5
3 + 5 = 8
5 + 8 = 13
8 + 13 = 21
13 + 21 = 34
21 + 34 = 55 
34 + 55 = 89
55 + 89 = 144
89 + 144 = 233
