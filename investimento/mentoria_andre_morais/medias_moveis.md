# Médias Móveis
Mais utilizado no mundo.
Vários indicadores são construídos a partir de médias móveis.
Rastrea a tendência do mercado.
Aritimética - mesmo peso para todos os dados coletados.
Exponecial - menos peso para os dados antigos e mais peso para os dados recentes.
A média funciona com um imã.
Calibragens usadas.
    * Exponencial de 17 e 72
    * Aritimética de 21 e  200
Identifica o início ou o término de uma tendência.
Reage a movimentação dos preços.
Não antecipa o movimento.
Médias de preços.
Médias de volume de negociação.
